## 方案

方案是这样子，我们实现跳动的CSS非常简单，无非是利用 keyframes 给输入框内部的元素套上动画，但如何控制每一个输入框内部的字符，才是比较棘手的，不过也不算复杂，尝试下以下两种方案：

- 覆盖式的 div，悬浮在 input 之上，拷贝 input 的字符组成 div 的内部元素
- 直接使用可编辑的 div，控制其内部元素的输入样式即可
